Excel File Format & Accepted Column Names

The importer accepts several common header names. Provide at least a name column.

Name — accepted: Full name, Name, name (required)
Email — accepted: Email address, Email, email
Phone — accepted: Phone number, Phone, phone
Skills / Areas — accepted: Skill area 1, Skill area 2, Skill area 3, or a combined column like Area of Expertise (comma/semicolon/newline separated)
Location — accepted: Location, location
BGE Code — accepted: BGE code, BGE Code, bge_code
Years of Experience — accepted header like Years of Experience
MSMEs Supported — optional count (header examples: MSMEs Supported)
Note: Only a name column is required. Phone numbers stored as numbers in Excel will be converted to strings; if phone is missing the field will be left blank.
